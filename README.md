# 💳 MerchantGuard

> A Playwright-based automation project for testing payment workflows, merchant operations, and API integrations.

MerchantGuard is inspired by real-world FinTech applications and focuses on validating critical business flows such as merchant onboarding, transactions, user management, and webhook integrations. The goal is to combine modern UI automation with API validation using Playwright.

---

## 🚀 Tech Stack

* 🎭 Playwright
* 📘 TypeScript
* 🔌 Playwright API Testing
* 📦 Node.js
* ⚙️ GitHub Actions
* 📊 Allure Reports
